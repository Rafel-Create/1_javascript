- onchange : HTML요소가 변경될 때
- onclick : 사용자가 HTML 엘리먼트를 클릭할 때(버튼 등)
- onmouseover : 사용자가 HTML 요소 위로 마우스를 이동합니다
- onmouseout : 사용자가 HTMl 요소에서 마우스를 멀리 이동합니다
- onkeydown : 사용자가 키보드의 특정 키를 누릅니다
- onload : 브라우저가 페이지 로딩을 완료합니다

html이벤트는 브라우저가 수행하는 일이거나  사용자가 수행하는 일일 수 있다.
- 페이지 로딩이 완료되었습니다
- 입력필드가 변경되었습니다
- 버튼이 클릭되었습니다
