class와 object

-class는 객체를 만들기 위한 템플릿(객체에 대한 금형)이다.
-객체는 사물(object)
-car(object)는 properties(속성) 와 method를 갖는다.
-프로퍼티 예시 properties[car.name = x6, car.weight = 1t] 
-메써드 예시 method[car.start(), car.brake()]
-데이터가 많을수록 객체지향으로 프로그래밍 하면 사용이 편리하다.
