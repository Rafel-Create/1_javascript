Operators

- 할당 연산자 =는 값을 할당합니다.  ex) let x = 10;
- 덧셈 연산자 +는 값을 더합니다.    ex) let x=5;let y=2; let z=x+y
- 곱셈 연산자 *는 값을 곱합니다.    ex) let q=5;let w=5; let multi= q*w
- 비교 연산자 >는 값을 비교합니다.  변수를 선언한 다음에 값을 비교
자바스크립트 연산자의 유형
산술연산자[+,-,*,**,/,%,++,--],할당연산자,비교연산자,문자열연산자,논리연산자,비트연산자,삼항연산자,

비교연산자

- 비교연산자는 결과값을 true와 false로 나눠준다.
- == 같음인데 엄격하게 같음은 아니다.

Logical operators : 논리연산자


- && => 그리고 조건이 두 개가 일치해야 true
- || => or 둘 중에 하나라도 맞으면 true
- ! => 같지 않음. 그래서 부정이 진실이고 진실이 거짓이다.
