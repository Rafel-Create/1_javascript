Conditional Statements

 

조건문
- if : 지정된 조건이 참인 경우 실행할 코드블럭을 지정하는 데 사용된다.
syntax
if(condition){}
- else : 동일한 조건이 거짓인 경우 실행할 코드블록을 지정하는데 사용합니다.
- else if : 첫 번째 조건이 거짓인 경우 실행할 코드블록을 지정하는데 사용합니다.
- switch : 월화수목금토일처럼 조건이 여러 개일 때. 실행할 여러 대체 코드블록을 지정하는데 사용됩니다.

switch : 조건이 여러 개일 경우 깔끔한 코딩을 위해 사용합니다.

    코드블록
