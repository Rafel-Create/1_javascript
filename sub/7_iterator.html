반보가능항목

- 반복가능한 객체(배열)입니다.
- for ..of : 문자열의 요소를 반복할 수 있다 
- 일련의 요소를 반복하는 것을 의미합니다
- 배열: 내장배열[정해져있는 반복]과 외장배열[정해져있지 않은 배열을 사용할 때]
- 중복을 허용하지 않는다.

Set 집합

- 고유한 값의 컬렉션
- 각 값은 Set에서 한 번만 발생할 수 있다.

Map

- 키와 값을 쌍으로 가지고 있다.
- 키는 모든 데이터의 유형별 삽입순서를 기억합니다. 
- 맵은 원래의 삽입순서를 기억합니다. 
- 여러 개의 값을 가지고 있는데 Map의 값을 가져올 때는 get()이라는 매서드를 통해(사용하여) 가져온다.
